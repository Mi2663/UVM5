# Ассемблер и интерпретатор УВМ (Вариант 5)

## Описание проекта
Реализация ассемблера и интерпретатора для Учебной Виртуальной Машины (УВМ), вариант 5.

## Спецификация УВМ (Вариант 5)

### Команды:
1. **LOAD_CONST** - Загрузка константы на стек
   - Код A: 2
   - Размер: 2 байта
   - Поле B: Константа (10 бит, 0-1023)

2. **LOAD_MEM** - Чтение из памяти
   - Код A: 3
   - Размер: 4 байта
   - Поле B: Адрес (24 бита, 0-16777215)

3. **STORE_MEM** - Запись в память
   - Код A: 1
   - Размер: 2 байта
   - Поле B: Смещение (13 бит, -4096..4095)

4. **ROL** - Циклический сдвиг влево
   - Код A: 4
   - Размер: 1 байт
